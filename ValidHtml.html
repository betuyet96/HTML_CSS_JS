What makes HTML valid?

Basic rules are:

  1. To tell the browser that the HTML document is an HTML 5, start the document with the following line:
    <!DOCTYPE Html>
  2. Apart from the doctype line, all your HTML document must be wrapped inside the element:
    <html>
  3. The element carries additional information for the whole webpage is:
    <head>
  4. Other HTML elements can appear in the <head>, and only in the <head>:
    <link>
    <meta>
    <style>
  5. The element where we write all our content is:
    <body>
    Everything inside the <body> will be displayed in the browser window.
  6. The W3C provides a Markup Validation Service to check any HTML document for errors and warnings.

The basic elements in HTML page are:

  1. A text header, denoted using the <h1>, <h2>, <h3>, <h4>, <h5>, <h6> tags.
  2. A paragraph, denoted using the <p> tag.
  3. A horizontal ruler, denoted using the <hr> tag.
  4. A link, denoted using the <a> (anchor) tag.
  5. A list, denoted using the <ul> (unordered list), <ol> (ordered list) and <li> (list element) tags.
  6. An image, denoted using the <img> tag
  7. A divider, denoted using the <div> tag
  8. A text span, denoted using the <span> tag
